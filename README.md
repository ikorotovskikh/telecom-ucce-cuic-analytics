# telecom-ucce-cuic-analytics
The repository contains a production-grade collection of CUIC Report Definitions (SQL-based), advanced stored procedures, 
and analytical queries designed for a Tier-1 Telecom Contact Center environment powered by Cisco UCCE (Unified Contact Center Enterprise) 12.x.

These assets are engineered for the Microsoft SQL Server database schema (AWDB and HDS) and their replicas for tens of thousands of agents.
